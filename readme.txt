English: Encyclopedia / Glossary Lexicon with selflinking Title Entrys for Pages
German: SEO Lexikon welches zwecks interner Verlinkung vorhandene Beiträge automatisch "querverlinkt".


== Description ==

Eine A-Z Buchstabennavigation wo Beiträge innerhalb der definierten Lexikon Seite alle Einträge alphabetisch aufgelistet werden.
Zusatzfunktion: Kommen in einem Lexikon Beitrag Wörter vor, die schon als Lexikon Eintrag vorhanden sind, verlinkt es automatisch diesen Lexikon Eintrag. Das ist also eine interne Quervlinkung wie es für SEO's manchmal wichtig ist.


[Video Tutorial/Beschreibung](https://www.youtube.com/watch?v=_Ay5IM5ePQM)

== Installation ==

1. Upload `seo-lexikon` Folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Settings you find in the new "Seo-Lexikon" Menu Name
